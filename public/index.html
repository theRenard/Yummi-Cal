datepick responsive theme

Welcome to the new Yummical theme form datepick.

Datepick™ is, is surely one of the best datepicker for jQuery but its themes are a bit, err… outdated… So here's a new one!

What's new?

  • A well LESS commented file, ready to be styled.
  • A javascript snippet for font size adjusting on resize events.
  • A sample page
  • Nothing else

What sould you do?

  1. Download datepick package from its website -> datepick
  2. Find "jquery.datepick.css" file and replace it with this one.
  3. Add this snippet to your code and call it on page loading/window resizing events. 
    
function datePickerTextSize() {

	var fontRatio = 0.050;

	$('#your_pickers_here').each(function() {
		var datepicker = $(this).find('.datepick'),
		datepickerWidth = datepicker.width(),
		newFontSize = parseInt(datepickerWidth * fontRatio);

		$(this).css({'font-size' : newFontSize});

	});
}
  4. Add these lines when you call for .datepick(); 
    
$('#your_pickers_here').datepick({

  onShow: function() {
    $('.datepick').removeAttr('style');
  }

});
  5. You are done.
